# Coding Quiz App

An interactive quiz application with timed questions, instant feedback, and performance-based scoring system.

## Features
- Interactive quiz interface with clear rules
- 15-second countdown timer per question
- 10 coding-related multiple-choice questions
- Instant answer validation with visual feedback
- Correct answer highlighting on wrong selection
- Score tracking and calculation
- Performance-based feedback messages
- Progress indicator (X of Y questions)
- One-time answer selection (no changes allowed)
- Auto-disable on time expiration
- Retake quiz functionality
- Responsive and modern UI design

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript (ES6 Modules)
- Font Awesome Icons


```bash
   cd Quiz-App
```
3. Open `index.html` in your browser
